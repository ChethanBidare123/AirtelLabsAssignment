Setup Process : 
Open IntelliJ and Import the project from this repository
Clean And Install the Project 
Run the TestNG.xml file to run both the use cases

# Code is written in Java

Framework:
test-selenium.properties file has the key value for browser selection and url
Used POM  where TestBase has all the driver initializations and derived all BeforeMethod, AfterMethod , After Class.
UiPages package is having all the UI functionalities and also has constant class.
Drivers folder stores all the executable drivers files
Output folder stores the screenshot as per the Test Name

## Every method is declared with javadocs for better readability
